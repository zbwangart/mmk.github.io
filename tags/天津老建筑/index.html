# 天津老建筑

2025

北宁公园-百年园林的静谧时光

2025-09-12

它不似皇家园林的恢弘，却自有岁月沉淀的温润与从容。

这里曾是清末“庚子事件”后重建的“鉴水榭”，历经百年风雨，如今是市民心中最亲切的城市绿肺。步入园中，古树参天，浓荫如盖，仿佛瞬间穿越了喧嚣。蜿蜒的湖岸线勾勒出碧波荡漾的蒲口，画舫轻摇，划开一池倒影；九曲回廊连接亭台楼阁，朱红柱、青灰瓦，在晨光或夕阳下泛着柔和的光泽。
